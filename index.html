よくある質問

対応している画像形式は？

入力：HEIC / HEIF / JPEG / PNG / WebP / GIF / BMP / TIFF に対応。出力：JPEG / PNG / WebP から選べます。

HEICとは何ですか？

HEIC（High Efficiency Image Container）は、iPhoneやiPadで標準的に使われる画像形式です。WindowsやAndroidでは開けないことがあるため、JPEG等への変換が必要になります。

画像はサーバーにアップロードされますか？

いいえ。すべての処理はブラウザ内で行われます。画像データが外部に送信されることは一切ありません。

変換枚数に制限はありますか？

ありません。何枚でも無料で変換できます。

スマートフォンでも使えますか？

はい。レスポンシブ対応済みで、スマートフォンからもご利用いただけます。
